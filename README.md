Corn3D
------

[Corn3D](http://code.google.com/p/corn3d/) is a simple 3D game engine
designed for Windows/DirectX platform. More information will be added later.

### Development

*	Main Repo: [Hosted on Google Code](http://code.google.com/p/corn3d/)
